# 致謝

## Vue Vben Admin

本專案的 Monorepo 架構和工程化實踐深受 [Vue Vben Admin](https://github.com/vbenjs/vue-vben-admin) 的啟發。

Vue Vben Admin 是一個基於 Vue 3、Vite、TypeScript 的中後台管理系統範本，擁有完善的 Monorepo 架構設計、強大的共享設定方案以及優秀的工程化實踐。我們在以下方面借鑒了其設計理念：

- **Monorepo 架構**：Turborepo + pnpm workspace 的專案組織方式
- **共享設定**：統一的 Vite 設定、TypeScript 設定和 Lint 設定
- **共享套件**：@core、locales、stores、utils 等套件的設計模式
- **工程化規範**：ESLint、Prettier、Stylelint、Commitlint 的統一設定方案
- **Mock 服務**：基於 Nitro 的 Mock 後端方案
- **國際化方案**：vue-i18n 多語言支援的實現方式

在此特別感謝 Vue Vben Admin 團隊和所有貢獻者的開源精神！

::: tip 推薦
如果您正在尋找一套完整的中後台管理解決方案，強烈推薦 [Vue Vben Admin](https://github.com/vbenjs/vue-vben-admin)。
:::
