# 致谢

## Vue Vben Admin

本项目的架构设计和部分代码参考了 [Vue Vben Admin](https://github.com/vbenjs/vue-vben-admin)，一个基于 Vue 3、Vite、TypeScript 的中后台管理解决方案。

在此特别感谢 Vue Vben Admin 团队的出色工作，他们的项目为我们提供了以下方面的灵感和参考：

- **Monorepo 架构**：Turborepo + pnpm workspace 的项目组织方式
- **国际化方案**：基于 vue-i18n 的多语言加载机制
- **状态管理**：Pinia 持久化存储方案
- **共享包设计**：locales、stores、utils 等包的模块化设计
- **Vite 配置**：统一的 Vite 构建配置方案
- **代码规范**：ESLint、Prettier、Stylelint、Commitlint 等统一规范配置

> **仓库地址**：[https://github.com/vbenjs/vue-vben-admin](https://github.com/vbenjs/vue-vben-admin)
>
> **开源协议**：MIT License
