# NutUI

Mobile H5 app based on [NutUI](https://nutui.jd.com/) 4.x.

## Features

- NutUI component auto-import via unplugin-vue-components
- Global SCSS variables support
- NutUI icons library `@nutui/icons-vue`

## Run

```bash
pnpm dev:nutui
```

Default port: `5777`

## NutUI Components Used

| Component                | Usage              |
| ------------------------ | ------------------ |
| `nut-navbar`             | Top navigation bar |
| `nut-tabbar`             | Bottom tab bar     |
| `nut-card`               | Product card       |
| `nut-swiper`             | Carousel           |
| `nut-form` / `nut-input` | Login form         |
| `nut-avatar`             | User avatar        |
| `nut-grid`               | Grid layout        |
| `nut-button`             | Button             |
| `nut-cell`               | List item          |
