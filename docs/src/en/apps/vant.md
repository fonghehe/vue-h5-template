# Vant

Mobile H5 app based on [Vant](https://vant-ui.github.io/vant/) 4.x.

## Features

- Vant global registration `app.use(Vant)`
- Built-in API-style components (Toast, Dialog)
- Vant component auto-import

## Run

```bash
pnpm dev:vant
```

Default port: `5778`

## Vant Components Used

| Component                | Usage              |
| ------------------------ | ------------------ |
| `van-nav-bar`            | Top navigation bar |
| `van-tabbar`             | Bottom tab bar     |
| `van-card`               | Product card       |
| `van-swipe`              | Carousel           |
| `van-form` / `van-field` | Login form         |
| `van-image`              | Image display      |
| `van-cell`               | List item          |
| `van-button`             | Button             |
| `van-tag`                | Tag                |
| `van-empty`              | Empty state        |
| `van-back-top`           | Back to top        |
