# Vant

[Vant](https://vant-ui.github.io/vant/) 4.x ベースのモバイル H5 アプリ。

## 特徴

- Vant グローバル登録 `app.use(Vant)`
- 組み込み API スタイルコンポーネント（Toast、Dialog）
- Vant コンポーネント自動インポート

## 起動

```bash
pnpm dev:vant
```

デフォルトポート：`5778`

## 使用している Vant コンポーネント

| コンポーネント           | 用途                     |
| ------------------------ | ------------------------ |
| `van-nav-bar`            | トップナビゲーションバー |
| `van-tabbar`             | ボトムタブバー           |
| `van-card`               | 商品カード               |
| `van-swipe`              | カルーセル               |
| `van-form` / `van-field` | ログインフォーム         |
| `van-image`              | 画像表示                 |
| `van-cell`               | リストアイテム           |
| `van-button`             | ボタン                   |
| `van-tag`                | タグ                     |
| `van-empty`              | 空状態                   |
| `van-back-top`           | トップに戻る             |
